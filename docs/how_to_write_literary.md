# 文藝感寫作指南

寫散文、隨筆、個人反思時，用這套方法。

---

## Step 1：找到你的核心矛盾

開始寫之前，先回答：**這篇文章裡，什麼東西是無法調和的？**

不是「我想談 X 主題」，而是「X 和 Y 之間有張力」。

範例：

- 知道什麼是好的 ↔ 做不出好的東西
- 做著最愛的工作 ↔ 痛苦得想逃離
- 渴望被完全看見 ↔ 永遠看不見真實的對方

這個矛盾會貫穿全文，每一段都在回應它。

---

## Step 2：用「思緒展開」取代「結論陳述」

❌ 不要這樣寫：

> 我當時很猶豫，不知道該怎麼開頭。

✅ 要這樣寫：

> 我寫下秋天的黃昏，停住。有點老土，刪掉。十月的黃昏——十月在北方已經入冬，在南方還像夏天，這個字沒有確定的季節感。不行，再刪。

**讓讀者看見你思考的過程，不是思考的結果。**

---

## Step 3：把抽象詞轉成具體畫面

每次寫到情緒或狀態詞，問自己：**這看起來像什麼？**

| 抽象         | 具體                                                       |
| ------------ | ---------------------------------------------------------- |
| 我很焦慮     | 光標閃爍，在等著我                                         |
| 感情在消退   | 像一朵在真空中綻放的花，一旦氧氣湧入，花瓣就開始腐爛       |
| 我們太苛刻   | 在黑板上抄寫數學家已經證明的定理，粉筆劃過的流暢令人愉悅   |
| 他很完美主義 | 他在旁邊標注：參考東京物語，光線角度十五度傾斜，營造疏離感 |

---

## Step 4：用短句堆疊製造節奏

當你想強調某個時刻，把長句拆成一行一個念頭：

```
再等等
他說
分鏡還有幾個地方不對
而且現在
讓演員拍的話
感覺會很尷尬
我還沒準備好
再等等
```

這種節奏像呼吸，讓讀者慢下來。

---

## Step 5：用重複句式推進思考

當你有一個洞察要展開，用「不是...而是...」或類似結構重複：

```
不是在讀德勒茲
而是在表演讀德勒茲這個行為
不是在看塔可夫斯基
而是在表演看塔可夫斯基這個行為
```

句式重複，內容遞進。讀者會跟著你的節奏走。

---

## Step 6：螺旋式結構，不是線性結構

不要 A → B → C 一路講下去。

要繞著同一個問題轉，每轉一圈就深一層：

1. 現象（我刪掉了開頭）
2. 普遍化（我見過很多這樣的人）
3. 具體案例（我認識一個朋友...）
4. 反例（但他的學弟...）
5. 提問（為什麼懂得越多，做得越少？）
6. 嘗試定義（品味是什麼）
7. 翻轉（品味的另一面）
8. 核心洞察（衝動死於被觀看的瞬間）
9. 出路（竭盡全力的輕盈）

**每一層都在回應 Step 1 的矛盾。**

---

## Step 7：用日記體創造真實感

穿插日期標記，製造時間跨度：

```
2022年2月日記一則
回想起以前的工作方法
那是一種愉快的自殺式的寫作
```

```
2025年5月1日
回家收拾舊物，找到兩箱稿紙
```

這讓讀者覺得這是真實經歷，不是編出來的。

---

## Step 8：引用要推進思考，不是裝飾

❌ 不要這樣：

> 正如德勒茲所說：「...」（然後繼續講別的）

✅ 要這樣：

> 我想起張定浩的「竭盡全力的輕盈」。竭盡全力聽起來很沉重，輕盈聽起來很放鬆。創作，不是在兩者之間二選一，而是讓兩者同時存在。但這像是個悖論...

引用是跳板，不是終點。引完要接著展開。

---

## Step 9：結尾不給答案，給姿態

❌ 不要這樣結尾：

> 所以，我們應該學會接受不完美。

✅ 要這樣結尾：

> 這回，我沒有刪除開頭。我把它變成了完成品。也許這就夠了。

> 我們在永恆的相互誤解和重新發現中繼續相愛，就是愛的最高形式。

**結尾重新定義問題，而不是解決問題。**

---

## 寫完後檢查

- [ ] 核心矛盾是什麼？有沒有貫穿全文？
- [ ] 有沒有「思緒展開」的段落？
- [ ] 抽象詞都轉成具體畫面了嗎？
- [ ] 有沒有短句堆疊的節奏變化？
- [ ] 引用有在推進思考嗎？
- [ ] 結尾是開放的嗎？
