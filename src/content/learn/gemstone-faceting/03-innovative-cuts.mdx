## 快速說明

過去五百年，寶石切割的基本原理沒有太大改變：用平面刻面反射光線。

直到 1990 年代，一個新技術打破了這個傳統：**凹面切割**（Concave Faceting）。

### 什麼是凹面切割？

傳統刻面是平的，像鏡子一樣反射光線。

凹面刻面是彎曲的，向內凹陷，像一個個小碗。這些「碗」不只反射光線，還會**折射和聚焦**光線，產生傳統切工無法達到的視覺效果。

差別就像：
- 傳統切工：手電筒照到鏡子，光線彈回來
- 凹面切工：手電筒照到放大鏡，光線被聚集、放大、分散

結果是什麼？寶石看起來像是**從內部發光**，而不是反射外部的光。

## Starbrite 切工

Starbrite 是美國寶石切割師 John Dyer 的註冊商標，代表他開發的一系列凹面切割設計。

### 為什麼 Starbrite 這麼特別？

傳統的圓明亮型切工有 57 個平面刻面。Starbrite 在亭部（底部）加入凹面的溝槽，這些溝槽：

1. **收集更多光線**：凹面像小型聚光器，把更多光線導入寶石內部
2. **分散光線**：光線在凹面內部折射後，從多個角度射出
3. **消除死區**：傳統切工在某些角度會出現「消光」（暗區），Starbrite 的凹面設計減少了這個問題

視覺效果：無論你從什麼角度看，寶石都在閃爍，而且亮度比傳統切工高 25-100%。

### John Dyer 是誰？

John Dyer 是當代最重要的寶石切割創新者之一。他的工作室在美國科羅拉多州，專門切割高端彩色寶石。

他的設計被博物館收藏，也常出現在高端珠寶設計中。如果你在珠寶展上看到一顆「不可能那麼亮」的彩色寶石，很可能就是他或他學生的作品。

Dyer 對自己的技術毫無保留。他出版了教學書籍、錄製教學影片，還設計了專門用於凹面切割的機器配件。這讓更多切割師能學習這種技術，但也代表「Starbrite」這個詞專指他的特定設計，其他人的凹面切割有各自的名稱。

## 凹面切割的種類

Starbrite 只是凹面切割的一種應用。這種技術可以用在不同部位：

| 應用方式 | 效果 |
|---------|------|
| 亭部凹面（如 Starbrite） | 最常見，增加整體亮度和火彩 |
| 冠部凹面 | 創造獨特的表面紋理，像水面波紋 |
| 全凹面 | 整顆寶石都用凹面，極致的光線效果 |
| 混合切工 | 部分傳統刻面 + 部分凹面，平衡亮度和經典外觀 |

## 為什麼不是所有寶石都用凹面切割？

既然這麼厲害，為什麼市場上大部分寶石還是傳統切工？

### 1. 技術門檻高

凹面切割需要：
- 特殊設備（凹面切割盤）
- 更長的加工時間（是傳統切工的 2-3 倍）
- 更高的技術經驗

大多數商業切割工廠沒有這個能力。

### 2. 原石損耗大

凹面切割會「挖掉」更多材料，原石利用率比傳統切工低。對於昂貴的原石（如優質藍寶石），這代表成本大幅增加。

### 3. 消費者不認識

大多數人買寶石時，最在意的是克拉數和證書。「切工品質」已經是次要考量，「創新切工技術」更是陌生概念。

這代表願意為凹面切割付出溢價的市場很小。

### 4. 某些寶石不適合

有些寶石的特性不適合凹面切割：
- 硬度低的寶石（凹面邊緣容易磨損）
- 有大量內含物的寶石（凹面會放大瑕疵）
- 顏色很淺的寶石（凹面的複雜折射可能讓顏色更淡）

## 其他創新切工

除了凹面切割，還有一些值得認識的創新技術：

### Fantasy Cut（幻想切工）

打破對稱限制，用不規則的刻面創造獨特效果。每顆都是獨一無二的藝術品，但難以估價，因為沒有標準可比較。

### Laser Cut（雷射切割）

用雷射在寶石內部「雕刻」圖案，創造立體的內部結構。常見於水晶紀念品，但也開始應用於高端彩色寶石。

### Torus Cut（環面切工）

把寶石切成甜甜圈形狀，中間有洞。視覺效果獨特，但鑲嵌困難，屬於收藏級設計。

## 練習

### 情境

你在珠寶展上看到兩顆藍寶石，顏色、大小、淨度都差不多，但價格差了三倍。賣家說貴的那顆是「Starbrite 切工」。

你仔細觀察，試圖判斷這個說法是否合理。

### 任務

根據以下觀察結果，判斷哪些是 Starbrite 切工的特徵，哪些不是：

| 觀察 | 是 Starbrite 特徵嗎？ |
|-----|---------------------|
| 從正面看，亭部有明顯的放射狀溝槽 | |
| 整顆寶石沒有明顯的暗區，轉動時持續閃爍 | |
| 刻面都是平的，排列非常對稱 | |
| 亮度比旁邊同等級的藍寶石明顯高 | |
| 寶石很輕，感覺密度不夠 | |

**你的答案：**

```
1. 放射狀溝槽 → 是 / 不是
2. 沒有暗區 → 是 / 不是
3. 刻面都是平的 → 是 / 不是
4. 亮度明顯較高 → 是 / 不是
5. 寶石很輕 → 是 / 不是
```

---

### 參考答案

```
1. 放射狀溝槽 → 是
   Starbrite 的招牌特徵就是亭部的凹面溝槽

2. 沒有暗區 → 是
   凹面設計減少消光，讓寶石在各角度都保持亮度

3. 刻面都是平的 → 不是
   這是傳統切工的特徵，不是凹面切割

4. 亮度明顯較高 → 是
   凹面切割可以增加 25-100% 的亮度

5. 寶石很輕 → 不是
   切工類型不會改變寶石的密度。如果感覺密度不對，
   可能是材質問題（仿品或處理過的石頭），跟切工無關
```

**進階判斷**：如果賣家說是 Starbrite，可以問他是否有 John Dyer 的證書或標誌。Starbrite 是註冊商標，只有 Dyer 工作室出品的才能用這個名字。其他人的凹面切割，品質可能一樣好，但不能叫 Starbrite。
