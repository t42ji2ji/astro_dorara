## 這個單元你會學會做什麼

看到一顆寶石，能夠說出它是什麼切工，以及這種切工的特性（適合什麼寶石、優缺點）。

🟢 基礎 ｜ 約 20 分鐘

---

## 快速說明

不同的切工不只是形狀不同，更重要的是它們處理光線的方式不同。

選對切工，可以讓一顆寶石的顏色更飽和、火彩更強烈、缺陷更不明顯。選錯切工，就算是頂級原石也會顯得平庸。

### 最常見的五種切工

| 切工 | 外觀 | 最適合 | 特點 |
|-----|------|-------|------|
| 圓明亮型 | 圓形，57-58 個刻面 | 鑽石 | 火彩最強，標準化程度最高 |
| 公主方型 | 正方形，尖角 | 鑽石、白色寶石 | 現代感，火彩強 |
| 祖母綠型 | 長方形，切角 | 祖母綠、海藍寶 | 突顯顏色和淨度，火彩弱 |
| 橢圓形 | 橢圓 | 彩色寶石 | 顯大，手指顯長 |
| 墊形 | 方形圓角 | 彩色寶石 | 復古風，顏色集中 |

---

### 圓明亮型切工（Round Brilliant）

這是鑽石的「標準答案」。

1919 年，Marcel Tolkowsky 用數學計算出理想的比例：57 個刻面，特定的角度組合，讓光線進入後幾乎 100% 從正面反射出來。

**為什麼鑽石幾乎都用這種切工？**

鑽石的折射率高（2.42），用圓明亮型切工可以最大化「火彩」（把白光分解成彩虹）。其他切工會損失這個優勢。

**其他寶石能用嗎？**

可以，但效果不同。彩色寶石的重點通常是「顏色」而不是「火彩」，圓明亮型的強烈閃爍反而會干擾顏色的呈現。

---

### 祖母綠型切工（Emerald Cut）

長方形，四個角被切掉，刻面是階梯狀的長條形。

**為什麼祖母綠要用這種切工？**

祖母綠天生多內含物（瑕疵），傳統切工的強烈閃爍會讓內含物更明顯。祖母綠型切工用大面積的平面，產生「鏡廳效應」（hall of mirrors）——優雅的明暗交替，而不是碎閃。

另一個原因是結構：祖母綠的晶體容易在切割時碎裂，切掉四個尖角可以減少風險。

**適合什麼人？**

喜歡低調、優雅風格的人。這種切工不「閃」，但很「亮」，適合正式場合。

**注意**：因為刻面大而平，任何瑕疵和顏色不均都會很明顯。選擇祖母綠型切工的寶石，淨度要求更高。

---

### 公主方型切工（Princess Cut）

正方形，底部是尖的（像倒過來的金字塔），刻面很多。

**特點**：

1. 火彩強，接近圓明亮型
2. 原石利用率高（圓形切工會浪費很多材料）
3. 現代感，適合簡約設計

**風險**：

四個角是尖的，容易碎裂。鑲嵌時必須用爪子保護角落，或選擇包鑲。

---

### 橢圓形切工（Oval Cut）

圓明亮型的變體，拉長成橢圓形。

**優點**：

1. 同樣克拉數，看起來比圓形大
2. 戴在手上會讓手指顯得修長
3. 火彩接近圓明亮型

**問題**：

中間可能出現「蝴蝶結效應」（bow-tie effect）——一個暗色的蝴蝶結形狀橫跨寶石中央。這是光線反射的幾何問題，幾乎所有橢圓形切工都有，只是程度不同。

選購時，要特別看正面是否有明顯的暗區。

---

### 墊形切工（Cushion Cut）

正方形或長方形，但四個角是圓的，像一個枕頭。

**為什麼彩色寶石喜歡這種切工？**

圓潤的角落和較大的刻面會讓顏色集中在中央，看起來更飽和。同時保留一定的火彩，不會像祖母綠型那麼「靜態」。

墊形切工有很多變體，刻面數量和排列方式各有不同。選購時要看實物，光看參數無法判斷。

## 練習

### 情境

你走進一家珠寶店，看到五顆寶石擺在展示櫃裡。店員說每顆的價格都差不多，但你發現它們的切工完全不同。

根據以下描述，判斷每顆是什麼切工：

### 任務

| 編號 | 描述 | 你的判斷 |
|-----|------|---------|
| A | 圓形，非常閃，像迪斯可球一樣碎光閃爍 | |
| B | 長方形，四角被削掉，表面像鏡子一樣規則的明暗條紋 | |
| C | 正方形，角很尖，閃爍程度接近 A | |
| D | 橢圓形，中間有一條隱約的暗色帶 | |
| E | 方形但角是圓的，顏色看起來很集中飽滿 | |

**你的答案：**

```
A = ____________
B = ____________
C = ____________
D = ____________
E = ____________
```

---

### 參考答案

```
A = 圓明亮型切工（Round Brilliant）
    特徵：圓形 + 強烈碎閃

B = 祖母綠型切工（Emerald Cut）
    特徵：長方形 + 切角 + 階梯狀刻面造成的規則明暗

C = 公主方型切工（Princess Cut）
    特徵：正方形 + 尖角 + 高火彩

D = 橢圓形切工（Oval Cut）
    特徵：橢圓 + 蝴蝶結效應

E = 墊形切工（Cushion Cut）
    特徵：圓角方形 + 顏色集中
```

## 快速對照表

買寶石前，用這張表快速判斷切工是否適合：

| 如果你想要... | 選這種切工 | 避免這種切工 |
|--------------|-----------|-------------|
| 最大火彩 | 圓明亮型、公主方型 | 祖母綠型 |
| 突顯顏色 | 墊形、祖母綠型 | 圓明亮型 |
| 顯大 | 橢圓形、馬眼形 | 墊形、圓形 |
| 低調優雅 | 祖母綠型 | 公主方型 |
| 保護尖角 | 墊形、圓形 | 公主方型、馬眼形 |
