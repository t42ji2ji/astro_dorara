## 這個單元你會學會做什麼

用肉眼觀察一顆寶石，判斷它的切工是「優秀」、「普通」還是「差」，並說出具體原因。

🟡 進階 ｜ 約 20 分鐘

---

## 快速說明

評估切工品質不需要昂貴的儀器。你需要的是：

1. 一個光源（自然光最好，或白色 LED）
2. 知道要看什麼
3. 一點練習

### 切工品質的三個面向

| 面向 | 看什麼 | 為什麼重要 |
|-----|-------|----------|
| 亮度（Brilliance） | 從正面看，白光反射的強度 | 切工好的寶石會把大部分光線反射回你的眼睛 |
| 火彩（Fire） | 彩虹色的閃爍 | 特別是鑽石，火彩是價值的重要指標 |
| 閃爍（Scintillation） | 轉動時明暗的變化 | 讓寶石「活」起來，不像塑膠 |

一個優秀的切工會在這三者之間取得平衡。只有亮度沒有火彩會像玻璃，只有火彩沒有亮度會像廉價水晶。

---

## 檢查清單：七個觀察點

### 1. 暗區測試

把寶石放在白色背景上，從正面看。

**問自己**：寶石中間有沒有明顯的暗區？

- 好的切工：整顆均勻明亮，或只有輕微的明暗對比
- 差的切工：中間有一塊「死掉」的區域，不管怎麼轉都是暗的

暗區代表光線從那個角度「漏掉了」，沒有反射回來。這通常是亭部角度切錯的結果。

### 2. 窗戶效應（Window）

換個角度：把寶石舉高，讓光線從背後穿過。

**問自己**：能不能透過寶石看到後面的東西？

- 好的切工：看不透，光線都被反射回來了
- 差的切工：中間像一扇窗戶，可以看到背後的桌面或手指

窗戶效應是亭部太淺的典型問題，常見於商業級彩色寶石。

### 3. 魚眼效應（Fish-eye）

這是窗戶效應的極端版本。

從正面看，如果你在寶石中央看到一個圓形的反射（像魚的眼睛），那就是亭部角度嚴重不足。

這種寶石基本沒有價值，應該避免購買。

### 4. 對稱性

觀察寶石的輪廓和刻面：

**問自己**：

- 圓形寶石真的是圓的嗎？還是有點歪？
- 對稱的刻面大小一樣嗎？
- 桌面有沒有偏離中心？

輕微的不對稱肉眼很難察覺，但嚴重的不對稱會影響光線表現，也代表切割師的技術不足。

### 5. 刻面銳利度

把寶石拿近，看刻面的邊緣：

**問自己**：刻面的交界線是銳利的直線，還是模糊的、圓潤的？

- 好的切工：邊緣像刀片一樣銳利
- 差的切工：邊緣模糊，或有「雙線」（代表拋光不完全）

### 6. 拋光品質

刻面表面應該像鏡子一樣光滑：

**問自己**：刻面上有沒有可見的刮痕、凹坑或波紋？

在強光下旋轉寶石，觀察反射的光線是否平順。如果看到「顫抖」的反射，代表拋光品質不佳。

### 7. 比例協調

最後，退後一步，整體看這顆寶石：

**問自己**：這顆寶石的各個部分看起來協調嗎？

- 桌面太大：火彩減少，看起來「空洞」
- 桌面太小：亮度不足，看起來「悶」
- 冠部太高：重量浪費在看不到的地方
- 冠部太低：火彩減少
- 亭部太深：從側面看太厚，正面看起來比實際小

## 練習

### 情境

你在二手市場看到三顆藍寶石，賣家說都是「天然藍寶石，有證書」。價格分別是 A 最貴、B 中等、C 最便宜。

你用這個單元學到的七點檢查每顆寶石，得到以下觀察結果：

### 觀察記錄

**寶石 A**
- 從正面看非常明亮，沒有暗區
- 舉高看，完全不透光
- 輪廓完美對稱
- 刻面邊緣銳利
- 表面像鏡子，沒有刮痕
- 整體比例協調

**寶石 B**
- 中間有一小塊暗區，但不明顯
- 舉高看，中央有輕微透光
- 輪廓大致對稱，但仔細看有點歪
- 刻面邊緣還算銳利
- 表面有幾條細微刮痕
- 看起來有點「厚」

**寶石 C**
- 中間有明顯的暗區
- 舉高看，可以清楚看到後面的手指
- 輪廓不太對稱
- 刻面邊緣有點模糊
- 表面拋光不均勻
- 桌面看起來特別大

### 任務

1. 根據觀察，為每顆寶石的切工評分（優秀 / 普通 / 差）
2. 說明你判斷的主要依據
3. 最便宜的 C 有什麼具體問題？

**你的答案：**

```
A 的評分：________
判斷依據：


B 的評分：________
判斷依據：


C 的評分：________
判斷依據：


C 的具體問題：


```

---

### 參考答案

```
A 的評分：優秀
判斷依據：七項檢查全部通過。沒有暗區、沒有窗戶效應、
完美對稱、銳利刻面、優質拋光、協調比例。這是精品級切工。

B 的評分：普通
判斷依據：有輕微的窗戶效應和暗區，對稱性稍差，有使用痕跡。
這是商業級切工，不差，但不是頂級。價格中等是合理的。

C 的評分：差
判斷依據：多項嚴重問題。

C 的具體問題：
1. 明顯的窗戶效應（亭部太淺，光線從底部漏掉）
2. 中央暗區（角度計算錯誤）
3. 不對稱（切割技術不足）
4. 刻面模糊 + 拋光不均（工序粗糙）
5. 桌面太大（可能是為了保留重量而犧牲比例）

這顆寶石即使是天然藍寶石，切工品質也讓它的視覺效果
大打折扣。除非價格非常低，否則不建議購買。
```

## 快速參考卡

下次看寶石時，記住這個順序：

```
1. 正面看 → 有沒有暗區？
2. 舉高看 → 有沒有窗戶？
3. 轉一圈 → 閃爍均勻嗎？
4. 看輪廓 → 對稱嗎？
5. 看細節 → 刻面銳利嗎？拋光好嗎？
6. 退後看 → 比例協調嗎？
```

如果前兩項就出問題，後面不用看了。

## 這堂課的結尾

你現在會了：

1. **說出寶石的五個部位**：桌面、冠部、腰圍、亭部、底尖
2. **區分常見切工**：圓明亮型、祖母綠型、公主方型、橢圓形、墊形
3. **理解創新切工**：凹面切割、Starbrite 的原理和視覺特徵
4. **評估切工品質**：用七點檢查法判斷好壞

### 最後的建議

理論學完了，接下來需要練習。

找機會去珠寶店或珠寶展，實際觀察不同切工的寶石。跟店員說你只是想看看，不買沒關係。大多數店員很樂意讓你比較不同的寶石。

看得越多，眼睛越準。

## 繼續學習

### 書籍

- **入門**：*Gemstones of the World* by Walter Schumann — 寶石圖鑑，有詳細的切工說明
- **進階**：*Secrets of the Gem Trade* by Richard W. Wise — 寶石商的實戰經驗，教你如何評估價值

### 線上資源

- **GIA（美國寶石學院）**：[gia.edu](https://www.gia.edu/) — 業界標準的寶石知識
- **John Dyer Gems**：[johndyergems.com](https://johndyergems.com/) — 凹面切割的作品集和技術文章
- **Gem Society**：[gemsociety.org](https://www.gemsociety.org/) — 寶石愛好者社群，有大量學習資源

### 實作

如果你對寶石切割產生興趣，可以從這裡開始：

- **書籍**：*Amateur Gemstone Faceting* by Tom Herbst — 業餘切割入門
- **設備**：入門級切割機約 $500-1000 美元
- **社群**：Facebook 群組 "Gem Cutters Guild" 有大量新手教學
