## 這個單元你會學會做什麼

看到一顆切割好的寶石，能夠指出它的**冠部、亭部、桌面、腰圍、底尖**在哪裡。

🟢 基礎 ｜ 約 10 分鐘

---

## 快速說明

在討論切工之前，你需要知道寶石的各個部位叫什麼。就像討論汽車性能之前，要先知道什麼是引擎、什麼是變速箱。

一顆標準的刻面寶石（faceted gemstone）分成五個部分：

### 結構圖

```
         ___________
        /   桌面    \        ← 桌面（Table）：最上方的平面
       /  ─────────  \       ← 冠部（Crown）：桌面到腰圍之間
      /               \
     |─────────────────|     ← 腰圍（Girdle）：最寬的地方，像腰帶
      \               /
       \             /       ← 亭部（Pavilion）：腰圍到底部
        \           /
         \    ▼    /
          \  底尖 /          ← 底尖（Culet）：最底部的點或小面
           \    /
            \  /
             \/
```

### 各部位的功能

| 部位 | 英文 | 功能 |
|-----|------|------|
| 桌面 | Table | 光線的主要入口，越大越亮，但火彩會減少 |
| 冠部 | Crown | 製造火彩（彩虹光），把白光分解成光譜 |
| 腰圍 | Girdle | 保護寶石，也是鑲嵌時固定的地方 |
| 亭部 | Pavilion | 反射光線的關鍵，角度決定了光線會不會「漏掉」 |
| 底尖 | Culet | 可以是尖點或小平面，影響光線反射路徑 |

### 為什麼亭部角度這麼重要？

這是切工的核心原理：

光線從桌面進入寶石，碰到亭部的刻面會反射。如果角度對了，光線會在亭部的兩個面之間反彈，然後從正面射出，進入你的眼睛。

角度太淺，光線從底部漏掉，寶石看起來暗淡，這叫**魚眼效應**（fish-eye）。
角度太深，光線從側面漏掉，中間出現黑色區域，這叫**消光**（extinction）。

每種寶石材質有不同的折射率，所以「正確的角度」也不同。鑽石是 40.75°，藍寶石是 42°，每一度的偏差都會影響光線表現。

## 練習

### 情境

你朋友剛買了一顆藍寶石戒指，興奮地拿給你看。她指著寶石問：「這個亮亮的平面是什麼？側邊這一圈又是什麼？」

### 任務

看下面這張示意圖，標出五個部位的名稱：

```
         ___________
        /     A     \
       /  ─────────  \
      /       B       \
     |────────C────────|
      \               /
       \      D      /
        \           /
         \         /
          \   E   /
           \     /
            \   /
             \ /
```

**你的答案：**

```
A = ____________
B = ____________
C = ____________
D = ____________
E = ____________
```

---

### 參考答案

```
A = 桌面（Table）
B = 冠部（Crown）
C = 腰圍（Girdle）
D = 亭部（Pavilion）
E = 底尖（Culet）
```

**記憶口訣**：從上往下，「桌冠腰亭尖」。

想像一個人站著：頭頂是桌子，戴著皇冠，繫著腰帶，下半身是亭子形狀，腳尖著地。

## 常見問題

### Q：為什麼有些寶石底部是平的，不是尖的？

有些切工會把底尖切成一個小平面（稱為 culet facet），這樣做可以：
1. 減少底尖被撞壞的風險
2. 降低某些角度的「針孔效應」

但太大的底尖平面會讓你從正面看到一個明顯的白點，影響美觀。現代鑽石通常會切成幾乎看不見的小點或完全尖銳。

### Q：腰圍可以做得很薄嗎？

可以，但風險是容易碎裂。鑲嵌師傅在固定寶石時施力的地方就是腰圍，太薄會裂開。所以好的切工會在「保留重量」和「結構安全」之間取得平衡。
