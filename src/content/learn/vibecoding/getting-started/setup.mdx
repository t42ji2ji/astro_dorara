在開始之前，讓我們先設定好開發環境。

## 推薦工具

### 編輯器
- **Cursor** - 內建 AI 功能的 VSCode fork
- **VSCode + Copilot** - 經典組合

### AI 服務
- **Claude** - 適合複雜推理和長文本
- **GPT-4** - 泛用性強
- **Claude Code** - CLI 工具，適合自動化

## 設定步驟

1. 安裝你選擇的編輯器
2. 設定 API key
3. 熟悉基本快捷鍵

完成設定後，我們就可以開始學習如何有效地使用這些工具了。
