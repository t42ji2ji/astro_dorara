# 搜尋策略指南

針對不同類型的課程主題，使用不同的搜尋策略。

---

## 技術工具類（Claude Code、Cursor、Excel）

**搜尋關鍵字組合**：

```
基礎：
- "{工具名}" official documentation
- "{工具名}" getting started tutorial 2024
- "{工具名}" tips and tricks

進階：
- "{工具名}" advanced features
- "{工具名}" vs {競品} comparison
- "{工具名}" workflow automation

中文：
- "{工具名}" 教學 繁體
- "{工具名}" 入門 指南
```

**優先來源**：
1. 官方文件（docs.xxx.com）
2. 官方部落格
3. YouTube 官方頻道
4. 知名技術部落格（CSS-Tricks、Smashing Magazine、Dev.to）

**避免**：
- 過時的 Stack Overflow 答案（看發布日期）
- 內容農場（標題黨、SEO 灌水文章）

---

## 方法論類（超速學習、原子習慣、Action Mapping）

**搜尋關鍵字組合**：

```
基礎：
- "{方法論}" summary key points
- "{方法論}" author interview
- "{方法論}" book review

應用：
- "{方法論}" case study
- "{方法論}" implementation example
- "{方法論}" template worksheet

學術：
- "{方法論}" research evidence
- "{方法論}" peer review study
```

**優先來源**：
1. 作者官網或部落格
2. 知名書評平台（Goodreads、豆瓣）
3. 學術資料庫（Google Scholar）
4. 知名 Podcast 訪談

**避免**：
- 純書摘網站（沒有自己的見解）
- 轉載文章（找原始來源）

---

## 技能類（Prompt Engineering、寫作、設計）

**搜尋關鍵字組合**：

```
基礎：
- "{技能}" beginner guide 2024
- "{技能}" fundamentals
- "{技能}" common mistakes

練習：
- "{技能}" exercises practice
- "{技能}" examples portfolio
- "{技能}" before after comparison

社群：
- "{技能}" community forum
- "{技能}" discord slack
- "{技能}" expert to follow
```

**優先來源**：
1. 業界專家個人部落格
2. 知名課程平台（Coursera、Udemy 高評分課程）
3. 作品集網站（Dribbble、Behance、GitHub）
4. 專業社群（Designer News、Hacker News）

**避免**：
- 沒有實際作品的「大師」
- 純理論沒有範例的文章

---

## 專業養成類（全端工程師、產品經理）

**搜尋關鍵字組合**：

```
路線圖：
- "{職業}" roadmap 2024
- "{職業}" learning path
- "{職業}" skills required

資源：
- "{職業}" free resources
- "{職業}" bootcamp comparison
- "{職業}" portfolio examples

職涯：
- "{職業}" interview preparation
- "{職業}" salary negotiation
- "{職業}" career transition
```

**優先來源**：
1. Roadmap.sh（技術路線圖）
2. 業界公司工程部落格（Stripe、Airbnb、Netflix）
3. 職涯平台（Levels.fyi、Glassdoor）
4. 專業社群（Reddit、Blind）

## AI/LLM 相關主題

**搜尋關鍵字組合**：

```
基礎：
- "{主題}" explained simply
- "{主題}" how it works
- "{主題}" getting started

實務：
- "{主題}" production use case
- "{主題}" implementation guide
- "{主題}" cost optimization

最新：
- "{主題}" 2025 updates
- "{主題}" latest developments
- site:arxiv.org "{主題}"
```

**優先來源**：
1. 官方文件（OpenAI、Anthropic、Google）
2. arXiv 論文（找易讀的解說文章）
3. AI 專門媒體（The Gradient、Towards Data Science）
4. AI 研究者 Twitter/部落格

**特別注意**：
- AI 領域變化快，注意發布日期
- 區分「行銷炒作」和「實際能力」
- 優先找有程式碼範例的文章

---

## 中文資源搜尋

當學員主要使用中文時，額外搜尋：

```
繁體中文：
- "{主題}" 繁體 教學
- "{主題}" 台灣 課程
- site:medium.com "{主題}" 中文

簡體中文（需驗證品質）：
- "{主題}" 知乎
- "{主題}" 掘金
- "{主題}" 极客时间
```

**可信的中文來源**：
- ExplainThis（前端/AI）
- PJCHENder（前端）
- 程式前沿（翻譯文章）
- iT 邦幫忙（技術社群）

---

## 搜尋品質檢核

每個搜尋結果要確認：

- [ ] 發布日期在可接受範圍內
- [ ] 作者有相關背景或作品
- [ ] 內容有實際範例或證據
- [ ] 不是純廣告或業配
- [ ] 連結可正常訪問
